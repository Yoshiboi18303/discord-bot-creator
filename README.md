# Discord Bot Creator

A creation CLI for quick creation of a Discord bot project.

---

## Quick Start

Make sure you have **[Node.js LTS/Latest](https://nodejs.org)** installed then run these commands:

```shell
npm i -g discord-bot-creator
```

```shell
create-discord-bot
```
